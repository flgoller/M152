Transformation

Mit Transformationen können elemente über CSS verschoben, vergrössert oder gedreht werden. Dazu wird die Eigenschaft "transformation" verwendet. Bei diesem beispiel wird das Div beim hovern um 90Grad gedreht. Dazu wird folgender Code verwendet: 'transform: rotate(90deg);'. Mit der eigenschaft Transition kann der Übergang des Zustandes beschreiben werden
